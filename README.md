# Final team project(ilhajob)
이 프로젝트는 구인구직 서비스를 제공하는 것을 목표로 구직을 원하는 일반회원과 구인을 원하는 기업회원의 롤을 나누어 제작하였습니다.

## 기술 스택
- Spring Boot 2.7         - Java 8         - Spring Data JPA
- spring MVC              - Oracle 12c     - apache tomcat
- thymeleaf               - jQuery         -  bootstrap
